# 🧠 Advanced Tic-Tac-Toe (Ultimate Tic-Tac-Toe)

เกม Tic-Tac-Toe แบบขั้นสูงที่มีกระดาน 3x3 ซ้อนกัน 9 ชั้น!

## 🎮 วิธีเล่น

### โครงสร้างเกม
- **กระดานใหญ่**: 3x3 (Big Board)
- **กระดานเล็ก**: แต่ละช่องของกระดานใหญ่เป็นกระดาน 3x3 อีกที
- **รวมทั้งหมด**: 9 กระดานเล็ก (81 ช่อง)

### กติกา
1. ผู้เล่นเริ่มที่กระดานไหนก็ได้
2. เมื่อวางในช่องย่อย → กำหนดกระดานที่ฝั่งตรงข้ามต้องไปเล่น
   - ตัวอย่าง: ถ้าเล่นช่อง (row 2, col 3) → ฝั่งตรงข้ามต้องไปเล่นในกระดานที่ 6
3. ถ้ากระดานนั้นเต็มหรือมีคนชนะแล้ว → เลือกเล่นกระดานไหนก็ได้

### เงื่อนไขชนะ
- **ระดับเล็ก**: ชนะเหมือน Tic-Tac-Toe ปกติ (3 เรียง)
- **ระดับใหญ่**: ชนะกระดานเล็ก = ยึดช่องนั้นในกระดานใหญ่
- **ชนะเกม**: ได้ 3 ช่องเรียงกันในกระดานใหญ่

## 🤖 โหมด AI

### 🟢 Easy
- สุ่มเลือกตำแหน่ง
- ไม่มีกลยุทธ์

### 🟡 Medium
- ตรวจสอบโอกาสชนะและบล็อก
- เลือกตำแหน่งสำคัญ (Center > Corner > Edge)
- มีการสุ่มเล็กน้อย

### 🔴 Hard
- ใช้ Heuristic Evaluation
- ประเมินทั้งกระดานเล็กและใหญ่
- คำนวณคะแนน:
  - ชนะ Big Board: +100
  - ชนะ Sub Board: +50
  - บล็อกคู่ต่อสู้: +40
  - ตำแหน่งกลาง: +5-10

## 📝 โครงสร้างโค้ด

```
game = {
  bigBoard: [9],        // สถานะกระดานใหญ่
  subBoards: [9][9],    // สถานะกระดานเล็กทั้ง 9 อัน
  currentPlayer: 'X',   // ผู้เล่นปัจจุบัน
  activeBoard: null,    // กระดานที่ต้องเล่น (null = เลือกได้ทุกกระดาน)
  gameMode: 'pvp',      // โหมดเกม
  difficulty: 'medium'  // ระดับ AI
}
```
